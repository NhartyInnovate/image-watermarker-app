# Image Watermarker Desktop App

A Tkinter + Pillow desktop application that allows users to add text watermarks to images.

## Features
- Upload image (PNG, JPG, JPEG)
- Add custom watermark text
- Choose watermark position
- Adjust opacity
- Preview before saving
- Save watermarked image

## Tech Stack
- Python
- Tkinter
- Pillow (PIL)

## How to Run

```bash
pip install pillow
python watermarker.py
